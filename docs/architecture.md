# Architecture

openhaze is a browserless, email-first AI sales agent. One binary runs the whole
funnel — **define ICP → discover → qualify → rank → find email → agentic email** — off
licensed data, with no social account and no scraping. It is a Rust port of
[`eracle/openhaze`](https://github.com/eracle/openhaze); this doc is the reader's
overview, and [PORTING.md](../PORTING.md) is the exhaustive module-by-module map.
